SANS — Viewer
=============

Requirements: Python 3 (https://python.org)
Python 3 comes pre-installed on most Macs and Linux systems.
On Windows, download from https://python.org/downloads/

HOW TO RUN
----------
Mac / Linux:
  1. Open Terminal in this folder
  2. Run: python3 serve.py
     (or double-click serve.sh if your system supports it)

Windows:
  1. Double-click serve.bat
     (or open Command Prompt here and run: python serve.py)

The viewer will open automatically in your browser at http://localhost:8888
Press Ctrl+C in the terminal to stop the server.

CONTROLS
--------
Click / Tap / Space / Enter / Right Arrow  — advance
Right-click / Left Arrow                   — go back

NOTES
-----
- Do not open index.html directly in your browser — audio won't work that way.
  Always use the server script above.
- Keep the terminal window open while viewing.
